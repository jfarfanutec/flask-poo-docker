# crud-application-using-flask-and-mysql
CRUD usando Flask y MySQL

#### componentes

* Python
* Python Libraries: flask and pymysql
* MySQL
* AdminLTE 2

#### ejecutar con Docker

```
docker-compose up -d
```

Al ejecutar tendremos 2 contenedores
contacto-app
contacto-mysql
http://ip:3000


para eliminar los contenedores, ejecutar:

```
docker-compose down --rmi all
```
